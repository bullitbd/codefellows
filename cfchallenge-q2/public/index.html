code challenge

"Create a simple http server using node. * This server should respond to a root-url request with a file called index.html. Do not use ExpressJS. Your code should have error checking and at least one callback. Put five or more html elements in your index.html. One of the elements should be a link to an external page."

This is an interesting exercise... having taken web servers for granted... even with the available abstractions (http, fs, path, etc), the process of thinking about all the edge cases & error handling (none of which is done here...) is challenging.
